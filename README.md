# Description

This module calculate Min, Max & Avg of an array

# Install

Go into a node project and type the following :

    $> npm install arraystat

# Exemple

    var arraystat = require('arraystat');

    console.log(arraystat([1,2,3,4,5])); // display : {max: 5, min: 1, avg: 3}

# Run

    $> node test.js
